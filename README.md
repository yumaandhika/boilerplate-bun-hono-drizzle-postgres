
## Fitur Utama

- **Rekomendasi Upgrade**: Memberikan saran upgrade untuk meningkatkan performa sistem.
- **Identifikasi Bottleneck**: Mendeteksi komponen yang membatasi performa sistem.
- **Pemeriksaan Kompatibilitas**: Memeriksa apakah komponen seperti prosesor dan motherboard, atau motherboard dan RAM, kompatibel.

## Teknologi

- **Hono**: Framework untuk membuat API.
- **Drizzle ORM**: ORM untuk berinteraksi dengan PostgreSQL.
- **Bun**: Alat untuk menjalankan dan mengelola proyek.
- **TypeScript**: Superset JavaScript untuk pengembangan yang lebih terstruktur.
- **PostgreSQL**: Database yang digunakan untuk menyimpan data.

To install dependencies:
```sh
bun install
```

To run:
```sh
bun run dev
```

open http://localhost:3000
